Clone the repository:
git clone [https://github.com/raconclips/studious-journey.git](https://github.com/raconclips/studious-journey.git)
cd crypto-tracker-cli


Install dependencies:
pip install -r requirements.txt


Run the tracker:
python tracker.py

Enjoy using! 😉




Tech Stack
​Language: Python 3
​Libraries: requests (for network calls), os & time (standard built-ins)
​API Target: CoinCap Rest API v2
